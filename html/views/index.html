What is it?

Infinity Scroll is a JavaScript library for creating truly infinite scroll lists of any size without browser limitations. While web browsers typically have a container height limit of 33,554,400px that restricts scrollable list sizes, Infinity Scroll solves this problem and enables working with lists containing millions of elements.

Key Features:

  • Truly infinite scroll — no limitations on the number of elements
  • Efficient DOM rendering — keeps only minimum necessary elements in DOM for optimal performance
  • Lazy data loading — fetches data from network only when needed, not all at once
  • Smart caching — saves loaded data in browser, eliminating repeated requests
  • Flexible data sources — works with local arrays and remote databases
  • Customizable templates — supports lists, tables, and custom div containers
  • Simple integration — minimal setup, maximum results

Perfect for:

• Large product catalogs
• News feeds and social media streams
• Search results with thousands of records
• Any interfaces with large datasets

Demo example:

Minimal setup:


// define config of your list
const MY_LIST_PROPS = {
  data: (start, end) => `http://jsonplaceholder.typicode.com/comments?_start=${start}&_end=${end}`,
  selectorId: 'MY_LIST',
  templateString: ({ item }) => `<li class="MY_LIST_Item" >
            ${item?.number} ${item?.name}
    </li>`,
};

// start it by call like new InfinityScroll(config)
new InfinityScroll(MY_LIST_PROPS);

How to use it:

1. Download InfinityScroll or copy remote link
2. Put script to the head tag or before your closing body tag
3. Place any tag with your unique ID into your html-code
4. Configure list in script tag. And then, start it by call new InfinityScroll(config)
