1) Data: object[] | string | (obj) => string;

data - your array with objects. Can be like:

  1. just array
  2. URL-string to your array in network
  3. Function, that's return URL-string to your array in network
    (obj: {
    start?: number;
    end?: number;
    page?: number;
    limit?: number;
    }) => string;     
data: BigJson1, // just array
// OR
data: 'https://jsonplaceholder.typicode.com/comments', // url-string
// OR
// function, returning URL-string (based on offset and limit)
data: ({ start, end, page, limit }) => `https://dummyjson.com/products?limit=${limit}&skip=${start}`,
// OR
// function, returning URL-string (based on start-end)
data: ({ start, end }) => `https://restapi.qoobeo.ru/api/v1/companys?start=${start}&end=${end}&isFullSchema=true`, //

2) SelectorId: string

ID of your root element

selectorId: 'myRootID';

3) TemplateString: (obj) => string

obj: {
item: { [key: string]: unknown };
listLength?: number;
idx?: number;
templateCb?: Record<string, (arg: any) => any>;
}

Template of list items

templateString: ({element, listLength}) => `<li class="REMOTE_LAZY_000ITEM_List__listItem big">
            ${element?.number} ${element?.name}
    </li>`,

4) ListWrapperHeight: number, (optional)

Height of your root element. Set in styles or in JS. 

listWrapperHeight: '350px',

5) ForcedListLength: number, (optional)

You can set length of your list manually. To limit very big list or just to accelerate initializating of list (to prevent automatical searching of list length). 

forcedListLength: 100,

6) ListType: 'list' | 'table' | 'div', (optional)

Switch type of your list between LIST (ul), TABLE (table) and just DIV (div). Default value is 'list'.
Attention: if you are using a TABLE, we strongly recommend setting the min-height of your TR (table row) based on the size of the maximum content that may be inside the cells. 

listType?: 'list';

7) tHeadNames: string[], (optional)

Title of your colums, while using table-mode intead of list 

tHeadNames: ['id', 'name', 'industry', 'founded', 'size', 'city (state)'],

8) SubDir: string, (optional)

Set additional directory to parce your API-response. In case, if you have response not like array, but like object with array 

const response = {
    status: 200,
    products: [{}, {}, {}, ...],
    total: 286,
    offset: 38,
};
      
subDir: 'products',
