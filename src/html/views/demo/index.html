Demo Configuration Types

Data Location:

• Local — data is stored on your device and loaded entirely at startup
• Remote — data needs to be downloaded from server/database

Loading Strategy:

• Batch/Full Load — downloads entire dataset at once (good for smaller datasets)
• On-Demand/Lazy Load — fetches data portions only when user scrolls to them (ideal for large datasets)


Note: Regardless of loading strategy, DOM rendering is always optimized — only visible elements are kept in memory.
