What is it?

This is a script to create really infinity scroll list (or another any scrollable entity). Unfortunately, web-browsers has limitation to show list of any height, so we have a limited presentation of list. This limit is 33554400px in height. But, working in progress and in version 2.0 we will solve this problem!

Demo list

Legend:
- Local/Remote - place of data (local data or remote DB)
- Instant/Lazy - type of load data. Instant load all db to memory. Lazy laod only visible parts.

  • Basic demo - Local Instant List (10 000 items) without styles
  • Simple demo - Local Instant List (100 items) with styles
  • Remote Instant List (500 items)
  • Remote Instant Api List (100 items)
  • Remote Lazy Api List (17 000 000+ items)
  • Remote Lazy Api List (100 items)

How to use it:

Description in progress:

const REMOTE_LAZY_000ITEMS_PROPS = {
  data: BigJson1,
  dataLoadPlace: 'remote',
  dataLoadSpeed: 'lazy',
  dataUrl: (start?: number, end?: number, page?: number, limit?: number) =>
    `http://localhost:3000/data?_start=${start}&_end=${end}`,
  name: 'my scroll list name',
  selectorId: 'REMOTE_LAZY_000ITEM',
  forcedListLength: 500,
  listType: 'list',
  listWrapperHeight: '290px',
  templateString: (element: object, listLength?: number): string => `<li
              class="REMOTE_LAZY_000ITEM_List__listItem big"
              aria-setsize="${listLength}"
              aria-posinset="${element?.number}"
      >
            ${element?.number} ${element?.name}
    </li>`,
};

new InfinityScroll(REMOTE_LAZY_000ITEMS_PROPS);
